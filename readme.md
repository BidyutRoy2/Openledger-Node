## Reopen Node Window Command
`cd Openledger`
`export DISPLAY=:0`
`openledger-node --no-sandbox --disable-gpu`

## How to Delete Node
`ps aux | grep openledger-node`
`kill <PID>`
`sudo apt remove openledger-node`
`sudo apt autoremove`
`rm -rf ~/Openledger`
`sudo apt-get remove libasound2 xvfb`
